# 🧑‍💻 Job Scraper using Selenium

## 📌 Overview
This project is an automated web scraper that extracts job listings from websites using Selenium.

## 🚀 Features
- Automated browser interaction using Selenium
- Extracts job title and company name
- Supports dynamic content handling
- Pagination for multiple pages
- Keyword-based filtering (Python jobs)
- Saves data into CSV using pandas

## 🛠 Tech Stack
- Python
- Selenium
- Pandas
- WebDriver Manager

## 📂 Project Structure
jobscrapper/
│── main.py
│── scraper/
│── utils/
│── data/
│── requirements.txt

## ▶️ How to Run

pip install -r requirements.txt
python main.py


## 📊 Output
- CSV file with job listings

## 💡 Use Case
Automates job data collection and eliminates manual searching.
